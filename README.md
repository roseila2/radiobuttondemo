Description

This is a simple Java GUI application that uses five radio buttons to allow the user to choose a pet.

The pets are:

Bird

Cat

Dog

Rabbit

Pig

When a pet is selected:

The picture of the pet is displayed.

A message box appears showing the name of the selected pet.

Tools Used

Java

Java Swing

Visual Studio Code (or any Java IDE)

How to Run

Download or clone the project.

Open the project in your Java IDE.

Compile and run the file:

javac RadioButtonDemo.java
java RadioButtonDemo
Project Files

RadioButtonDemo.java – Main program

images folder – Contains pet images

README.md – Project description
